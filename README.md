# Student Course Registration System

## Description

A command-line Python application used to manage students, courses, and course registrations.

## Features

- Add students
- View students
- Search students
- Add courses
- View courses
- Register students to courses
- Prevent duplicate registrations
- Prevent over-capacity registrations
- Save data to JSON files
- Load data from JSON files
- Inheritance using Person and Student classes

## Classes Used

- Person
- Student
- Course
- SchoolSystem

## How to Run

```bash
python main.py
```
